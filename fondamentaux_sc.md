# 2. Les 4 piliers de l’apprentissage

Référence : Stanislas Dehaene et les travaux en sciences cognitives.

## 2.1 Les quatre piliers

### 1. L’attention

L’attention sélectionne les informations à traiter. Sans attention, il n’y a pas d’encodage solide.  
L’enjeu pédagogique est donc de **guider l’attention** vers ce qui compte vraiment.

Implications :

- annoncer clairement l’objectif de la séance ;
- éviter les supports trop chargés ;
- signaler explicitement les éléments importants ;
- réduire les distractions ;
- découper les activités longues ;
- varier les modalités sans disperser les élèves ;
- faire reformuler ce qu’il faut regarder, écouter, retenir ou produire.

Erreur fréquente : croire qu’un élève exposé à une information l’a nécessairement traitée. En réalité, il peut être présent physiquement mais absent cognitivement.

### 2. L’engagement actif

L’apprentissage augmente quand l’élève fait quelque chose mentalement : chercher, prédire, comparer, expliquer, résoudre, s’auto-questionner.

Un élève actif n’est pas seulement un élève qui bouge ou qui parle. L’activité importante est **l’activité cognitive**.

Bonnes pratiques :

- poser une question avant d’apporter la réponse ;
- demander une hypothèse ;
- faire résoudre un mini-problème ;
- faire classer, comparer ou justifier ;
- intégrer des tests courts ;
- demander une auto-explication ;
- laisser un temps de recherche réel.

### 3. Le retour d’information / feedback

L’erreur est utile si elle est identifiée, comprise et corrigée. Le feedback permet de réduire l’écart entre ce que l’élève produit et ce qui est attendu.

Un feedback efficace est :

- proche dans le temps ;
- spécifique ;
- centré sur la tâche ou la stratégie, pas sur la personne ;
- orienté vers une amélioration concrète ;
- compréhensible par l’élève ;
- suivi d’un temps de reprise.

À éviter :

- « C’est faux » sans explication ;
- la note seule ;
- les corrections trop tardives ;
- les commentaires trop longs jamais retravaillés ;
- les feedbacks qui jugent l’élève plutôt que son action.

### 4. La consolidation

Une connaissance fragile disparaît vite si elle n’est pas réactivée. La consolidation demande du temps, de la répétition, de l’espacement et de la récupération active.

Principes essentiels :

- revoir plusieurs fois ;
- espacer les reprises ;
- tester sans forcément noter ;
- mélanger les anciennes et nouvelles notions ;
- faire rappeler de mémoire avant de relire ;
- automatiser les essentiels pour libérer la mémoire de travail.

Conclusion pratique : un cours réussi n’est pas seulement un cours clair. C’est un cours qui prévoit déjà les moments où les élèves devront **réactiver** ce qu’ils ont appris.

# 3. Les types de connaissances

Le Digipad s’appuie notamment sur les travaux d’André Tricot.

## 3.1 Connaissances primaires et secondaires

### Connaissances primaires

Ce sont des connaissances que l’humain acquiert relativement naturellement dans l’environnement quotidien : parler sa langue maternelle, reconnaître des visages, interagir socialement, comprendre des situations ordinaires.

Elles demandent peu d’enseignement explicite, car elles se construisent par immersion et interaction.

### Connaissances secondaires

Ce sont les connaissances scolaires : lire, écrire, calculer, raisonner scientifiquement, produire une dissertation, lire une carte, comprendre un concept abstrait.

Elles ne s’acquièrent pas spontanément. Elles exigent :

- attention volontaire ;
- effort ;
- guidage ;
- entraînement ;
- feedback ;
- motivation ;
- consolidation.

## 3.2 Conséquence majeure pour l’enseignement

L’école demande principalement d’acquérir des connaissances secondaires. Il est donc normal que les élèves aient besoin d’un enseignement structuré, explicite et répété.

Ce point permet de lutter contre une idée dangereuse : « s’il n’a pas compris, c’est qu’il n’a pas écouté » ou « s’il ne sait pas faire, c’est qu’il ne veut pas ». Souvent, l’élève n’a pas encore construit les prérequis, les stratégies ou la représentation de la tâche.

## 3.3 Autres distinctions utiles

### Connaissances déclaratives

Savoirs que l’on peut formuler : définitions, faits, dates, règles, notions.

Exemple : savoir définir une métropole, connaître la règle d’accord du participe passé, connaître la formule de l’aire d’un triangle.

### Connaissances procédurales

Savoir-faire : appliquer une méthode, résoudre un problème, rédiger une réponse, construire un graphique.

Exemple : savoir analyser un document, savoir utiliser une formule, savoir rédiger une introduction.

### Connaissances conditionnelles

Savoir quand et pourquoi utiliser une connaissance ou une procédure.

Exemple : savoir quand utiliser tel type de graphique, quand justifier par un exemple, quand mobiliser telle méthode de calcul.

## 3.4 Implication pédagogique

Une séquence complète doit travailler les trois niveaux :

- **Quoi ?** Les notions et informations à connaître.
- **Comment ?** Les procédures pour les utiliser.
- **Quand / pourquoi ?** Les situations dans lesquelles les mobiliser.

Beaucoup d’échecs scolaires viennent du fait que l’enseignant a enseigné le « quoi », parfois le « comment », mais rarement le « quand ».

# 4. Les fonctions de la consigne

Référence mentionnée : Philippe Meirieu.

La consigne n’est pas un simple ordre. Elle structure l’activité cognitive de l’élève. Une mauvaise consigne peut produire de l’échec même lorsque l’élève possède les connaissances nécessaires.

## 4.1 Les fonctions principales

### 1. La consigne-but

Elle dit ce qu’il faut atteindre.

Exemples :

- « Explique pourquoi cette ville est une métropole. »
- « Rédige un paragraphe argumenté. »
- « Résous ce problème en justifiant ta démarche. »

Risque : si le but est trop vague, les élèves ne savent pas ce qui compte vraiment.

### 2. La consigne-procédure

Elle indique comment faire.

Exemples :

- « Lis d’abord le document 1, puis relève deux informations. »
- « Commence par identifier les données utiles. »
- « Utilise la méthode : je repère, je cite, j’explique. »

Elle est essentielle pour les élèves qui ne maîtrisent pas encore les stratégies expertes.

### 3. La consigne-structure

Elle organise le travail : étapes, temps, forme, supports, modalités.

Exemples :

- « Travail individuel pendant 5 minutes, puis comparaison en binôme. »
- « Réponds sous forme de tableau. »
- « Utilise trois couleurs : causes, conséquences, exemples. »

### 4. La consigne-critère

Elle précise les critères de réussite.

Exemples :

- « Ta réponse doit contenir une idée, une preuve tirée du document et une explication. »
- « Le schéma est réussi si les flèches sont nommées et si les relations sont compréhensibles. »

C’est une consigne décisive pour rendre l’évaluation moins implicite.

### 5. La consigne-vérification

Elle invite l’élève à contrôler son travail.

Exemples :

- « Relis ta réponse : as-tu cité le document ? »
- « Vérifie que ton résultat est cohérent avec l’unité demandée. »
- « Compare ta production avec les critères de réussite. »

## 4.2 Checklist d’une bonne consigne

Une consigne solide répond à cinq questions :

1. Quel est le but ?
2. Quelles étapes suivre ?
3. Quelle forme de réponse produire ?
4. Quels critères permettent de réussir ?
5. Comment vérifier avant de rendre ?

# 5. Enseigner plus explicitement

Le Digipad précise qu’il ne s’agit pas uniquement de « pédagogie explicite » comme méthode fermée, mais d’une démarche plus large : **lever les malentendus et les implicites**.

## 5.1 Le problème des implicites scolaires

Les enseignants savent souvent ce qu’ils attendent, mais les élèves ne le devinent pas toujours.

Exemples d’implicites :

- ce qu’il faut apprendre précisément ;
- ce qui est important dans un document ;
- la différence entre lire et étudier ;
- ce que signifie « justifier » ;
- comment réviser efficacement ;
- comment présenter une réponse ;
- comment transférer une méthode dans un nouveau contexte.

Les élèves les plus proches des codes scolaires décodent ces implicites. Les autres peuvent travailler beaucoup sans comprendre ce qui est réellement attendu.

## 5.2 Enseigner explicitement, c’est rendre visible

À rendre visible :

- les objectifs ;
- les savoirs essentiels ;
- les procédures ;
- les critères de réussite ;
- les erreurs fréquentes ;
- les stratégies efficaces ;
- les liens entre activités et apprentissages ;
- les raisons des choix pédagogiques.

## 5.3 Structure utile d’une séance explicite

1. **Objectif clair** : « Aujourd’hui, on apprend à… »
2. **Activation des prérequis** : rappel rapide de ce qui est nécessaire.
3. **Modélisation** : montrer comment un expert fait.
4. **Guidage** : faire avec les élèves.
5. **Pratique active** : laisser les élèves faire.
6. **Feedback** : corriger et ajuster.
7. **Institutionnalisation** : formuler ce qu’il faut retenir.
8. **Réactivation prévue** : annoncer quand et comment ce sera repris.

## 5.4 Phrase clé

> Ce qui est évident pour l’enseignant ne l’est pas forcément pour l’élève.

# 6. Définir le travail personnel de l’élève

Le Digipad insiste sur un point important : le travail personnel de l’élève ne se réduit pas au travail à la maison.

## 6.1 Définition élargie

Le travail personnel commence en classe : écouter, comprendre, s’engager, chercher, reformuler, s’entraîner, questionner, mémoriser, vérifier.

Le travail hors classe n’est qu’une partie du processus.

## 6.2 Les dimensions du travail personnel

### En classe

- attention ;
- participation cognitive ;
- prise d’informations ;
- premiers essais ;
- questionnement ;
- correction ;
- mémorisation initiale ;
- appropriation des méthodes.

### Hors classe

- réactivation ;
- entraînement ;
- consolidation ;
- préparation ;
- production longue ;
- auto-évaluation.

## 6.3 Risque majeur

Donner du travail à la maison sans avoir enseigné comment le faire revient à externaliser la difficulté vers les familles ou vers l’autonomie supposée de l’élève.

Pour être équitable, le travail personnel doit être :

- explicité ;
- préparé en classe ;
- relié à un objectif ;
- calibré ;
- vérifiable ;
- réinvesti ensuite.

## 6.4 Bonnes questions avant de donner un devoir

- Quel apprentissage ce devoir consolide-t-il ?
- L’élève sait-il comment s’y prendre ?
- A-t-on commencé la méthode en classe ?
- Le temps demandé est-il raisonnable ?
- Comment le travail sera-t-il repris ?
- Est-ce un travail de mémorisation, d’entraînement, de préparation ou de production ?

# 7. Différenciation pédagogique

La différenciation pédagogique vise à prendre en compte l’hétérogénéité sans renoncer aux objectifs communs.

## 7.1 Différencier ne veut pas dire individualiser tout le temps

Différencier ne signifie pas préparer 30 cours différents pour 30 élèves. C’est ajuster certains paramètres pour permettre à davantage d’élèves d’atteindre l’objectif.

## 7.2 Les grands objets de différenciation

### 1. Les contenus

Adapter ou varier les supports, les documents, les niveaux de guidage, les quantités d’information.

Exemples :

- documents plus ou moins longs ;
- lexique fourni ;
- version audio ;
- texte annoté ;
- étapes intermédiaires.

### 2. Les processus

Varier les chemins pour apprendre.

Exemples :

- travail individuel puis binôme ;
- manipulation ;
- carte mentale ;
- guidage progressif ;
- tutorat ;
- choix entre deux méthodes.

### 3. Les productions

Permettre différentes formes de restitution lorsque l’objectif le permet.

Exemples :

- paragraphe ;
- schéma ;
- oral ;
- carte mentale ;
- capsule audio ;
- tableau comparatif.

### 4. Les structures

Adapter l’organisation matérielle ou sociale.

Exemples :

- groupes de besoin ;
- ateliers ;
- classe flexible ;
- plans de travail ;
- temps différenciés ;
- aide ponctuelle ciblée.

## 7.3 Différenciation a priori et a posteriori

### A priori

Prévoir les obstacles avant la séance.

Exemples :

- anticiper le vocabulaire difficile ;
- prévoir une aide méthodologique ;
- proposer un support gradué ;
- préparer un exemple modèle.

### A posteriori

Adapter après observation des difficultés.

Exemples :

- groupe de reprise ;
- feedback ciblé ;
- exercice supplémentaire ;
- remédiation sur une erreur fréquente.

## 7.4 Principe de vigilance

Différencier ne doit pas enfermer les élèves faibles dans des tâches pauvres. L’aide doit réduire les obstacles, pas diminuer l’ambition intellectuelle.

# 8. Amener les élèves à apprendre par leur travail personnel

Cette partie relie travail personnel, explicitation, mémorisation et autonomie.

## 8.1 L’autonomie s’enseigne

On demande souvent aux élèves d’être autonomes, mais on enseigne rarement les gestes concrets de l’autonomie.

Gestes à enseigner :

- planifier ;
- relire activement ;
- se tester ;
- identifier ce qui est su ou non su ;
- demander de l’aide précisément ;
- corriger une erreur ;
- utiliser une fiche de mémorisation ;
- reprendre un feedback.

## 8.2 Apprendre à apprendre

Apprendre à apprendre ne doit pas être un slogan. Cela suppose des outils concrets :

- fiches de mémorisation ;
- calendrier de reprises ;
- rituels de rappel ;
- critères de réussite ;
- exemples corrigés ;
- entraînement guidé ;
- verbalisation des stratégies.

## 8.3 L’entretien d’explicitation

Le Digipad associé contient des ressources sur l’entretien d’explicitation de Pierre Vermersch.

Idée centrale : aider l’élève à revenir sur son action réelle, pas seulement sur ce qu’il pense avoir fait.

Questions utiles :

- « Comment t’y es-tu pris ? »
- « Qu’as-tu fait en premier ? »
- « À quel moment as-tu hésité ? »
- « Qu’est-ce qui t’a fait choisir cette réponse ? »
- « Comment pourrais-tu t’y prendre la prochaine fois ? »

Objectif : développer la métacognition, c’est-à-dire la capacité à comprendre ses propres démarches.

# 9. Les stratégies de mémorisation

## 9.1 Idée centrale

Relire plusieurs fois donne une impression de familiarité, mais ce n’est pas la stratégie la plus efficace pour mémoriser durablement.

Les stratégies les plus utiles reposent sur :

- la récupération active ;
- l’espacement ;
- l’entraînement répété ;
- l’élaboration ;
- l’interleaving, c’est-à-dire l’alternance de notions ou types d’exercices.

## 9.2 Récupération active

Se tester de mémoire est plus efficace que relire passivement.

Exemples :

- cacher le cours et répondre à des questions ;
- utiliser des flashcards ;
- faire un quiz ;
- écrire tout ce dont on se souvient ;
- expliquer à quelqu’un ;
- refaire un exercice sans modèle.

## 9.3 Espacement

Mieux vaut 4 reprises courtes espacées qu’une longue session la veille.

Exemple de calendrier :

- J0 : découverte de la notion ;
- J1 : rappel court ;
- J3 : quiz rapide ;
- J7 : exercice de réinvestissement ;
- J14 : rappel mélangé avec une autre notion ;
- J30 : évaluation ou réactivation longue.

## 9.4 Feedback et correction

Se tester sans corriger peut renforcer l’erreur. Il faut donc prévoir :

- une correction fiable ;
- une explication de l’erreur ;
- une nouvelle tentative ;
- une reprise plus tard.

## 9.5 Outils mentionnés dans le Digipad de mémorisation active

- testing régulier ;
- répertoire des essentiels ;
- multitesting des essentiels ;
- indices de rappel ;
- encarts de mémorisation ;
- fiches de mémorisation ;
- cartes de mémorisation / flashcards ;
- cahier de réactivation ;
- calendrier de reprises ;
- outils numériques comme Quizlet, Anki, Plickers, Kahoot, AgoraQuiz, Édumalin.

# 10. Comparer l’efficacité des stratégies de mémorisation

## 10.1 Stratégies souvent peu efficaces seules

- relire passivement ;
- surligner sans objectif ;
- recopier le cours ;
- apprendre uniquement la veille ;
- regarder une correction sans refaire ;
- croire qu’on sait parce que le cours paraît familier.

Ces stratégies peuvent rassurer, mais elles ne garantissent pas la récupération future.

## 10.2 Stratégies plus efficaces

### Se tester

Le test n’est pas seulement un outil d’évaluation : c’est un outil d’apprentissage.

### Espacer les reprises

L’oubli partiel rend la récupération plus difficile, mais aussi plus bénéfique.

### Alterner les exercices

Mélanger plusieurs types de problèmes oblige à reconnaître quelle stratégie utiliser.

### Expliquer

Produire une explication oblige à organiser les connaissances.

### Utiliser des exemples et contre-exemples

Permet de mieux construire les catégories conceptuelles.

## 10.3 Message à transmettre aux élèves

> Si tu relis et que tu comprends, ce n’est pas encore gagné. Tu dois vérifier que tu peux retrouver l’information sans l’avoir sous les yeux.

# 11. Motivation scolaire

Le Digipad distingue plusieurs entrées : modalités de la motivation, théories de la motivation, et sentiment d’efficacité personnelle.

## 11.1 Les 4 modalités de la motivation

La ressource renvoie à une fiche-synthèse sur quatre modalités. En pratique pédagogique, on peut les rapprocher de quatre grands leviers :

### 1. Le sentiment de compétence

L’élève s’engage davantage s’il pense pouvoir réussir.

À renforcer par :

- objectifs atteignables ;
- progression visible ;
- feedbacks précis ;
- réussites graduées ;
- droit à l’erreur ;
- modèles de réussite.

### 2. La valeur perçue de la tâche

L’élève s’engage si la tâche a du sens, de l’utilité ou de l’intérêt.

À renforcer par :

- expliciter le pourquoi ;
- relier à des situations concrètes ;
- montrer l’usage futur ;
- proposer des problèmes authentiques ;
- donner une part de choix.

### 3. Le contrôle / l’autonomie

L’élève est plus motivé s’il a une marge de décision réelle.

À renforcer par :

- choix du support ;
- choix de stratégie ;
- choix d’ordre des tâches ;
- auto-évaluation ;
- objectifs personnels.

### 4. L’appartenance / climat relationnel

La motivation est influencée par la relation à l’enseignant, au groupe, et par le sentiment d’être reconnu.

À renforcer par :

- climat sécurisant ;
- coopération ;
- encouragements crédibles ;
- absence d’humiliation ;
- feedbacks respectueux ;
- valorisation des progrès.

## 11.2 Les 5 théories de la motivation mentionnées

Le Digipad renvoie à une fiche sur 5 théories de la motivation. Les ressources associées mentionnent notamment Franck Ramus, Fabien Fenouillet, la métacognition et les leviers motivationnels.

Approche synthétique utilisable :

### 1. Théorie de l’autodétermination

La motivation augmente quand trois besoins sont nourris :

- autonomie ;
- compétence ;
- appartenance sociale.

### 2. Théorie attentes-valeur

L’élève s’engage s’il pense pouvoir réussir et si la tâche vaut l’effort.

Deux questions clés :

- « Est-ce que je peux y arriver ? »
- « Est-ce que ça vaut le coup ? »

### 3. Sentiment d’efficacité personnelle

L’élève agit selon sa croyance dans sa capacité à réussir une tâche précise.

### 4. Buts d’accomplissement

Certains élèves cherchent à apprendre et progresser ; d’autres cherchent surtout à paraître compétents ou à éviter de paraître faibles.

Un climat trop centré sur la comparaison peut favoriser l’évitement.

### 5. Attribution causale

L’élève interprète ses réussites et échecs : effort, capacité, chance, difficulté, aide reçue.

Objectif : l’aider à attribuer ses progrès à des causes contrôlables : stratégie, entraînement, attention, correction.

# 12. Sentiment d’efficacité personnelle — S.E.P.

Référence centrale : Albert Bandura.

## 12.1 Définition

Le sentiment d’efficacité personnelle correspond à la croyance d’un individu dans sa capacité à organiser et exécuter les actions nécessaires pour réussir une tâche.

Ce n’est pas une confiance générale. C’est une confiance située : « je pense pouvoir réussir cette tâche précise dans ces conditions ».

## 12.2 Les 4 sources du S.E.P.

### 1. Les expériences de maîtrise

C’est la source la plus puissante. Réussir une tâche renforce l’idée qu’on peut réussir à nouveau.

À favoriser par :

- tâches graduées ;
- objectifs atteignables ;
- progression visible ;
- entraînement ;
- réussites répétées.

### 2. Les expériences vicariantes

Voir quelqu’un de proche de soi réussir peut renforcer la croyance qu’on peut réussir aussi.

À favoriser par :

- exemples d’élèves ;
- correction commentée ;
- modelage ;
- tutorat ;
- pair qui explique sa démarche.

### 3. La persuasion verbale

Les encouragements peuvent aider s’ils sont crédibles, précis et liés à des actions.

Bon exemple :

- « Ta méthode de repérage fonctionne mieux qu’hier ; reprends-la sur le document 2. »

Moins bon exemple :

- « Tu es super, crois en toi. »

### 4. Les états physiologiques et émotionnels

Stress, fatigue, peur de l’échec ou sentiment d’insécurité diminuent la perception de compétence.

À favoriser :

- climat d’erreur sécurisé ;
- temps suffisant ;
- consignes claires ;
- routines ;
- évaluations préparées ;
- droit à l’essai.

## 12.3 Application directe

Pour renforcer le S.E.P., il ne suffit pas de motiver verbalement. Il faut organiser des situations où l’élève **réussit réellement**, comprend pourquoi il réussit, et peut transférer cette réussite.

# 13. Modèle opérationnel pour concevoir une séance efficace

## Étape 1 — Objectif

Formuler ce que les élèves doivent être capables de faire à la fin.

Exemple :

> Être capable d’expliquer une notion en utilisant un exemple et une preuve tirée d’un document.

## Étape 2 — Pré-requis

Identifier ce qui doit déjà être connu.

- vocabulaire ;
- méthode ;
- notion antérieure ;
- procédure technique.

## Étape 3 — Attention

Prévoir comment orienter l’attention.

- question-problème ;
- annonce des critères ;
- support épuré ;
- guidage visuel ;
- temps courts.

## Étape 4 — Engagement actif

Prévoir ce que les élèves feront mentalement.

- chercher ;
- prédire ;
- expliquer ;
- comparer ;
- classer ;
- produire ;
- se tester.

## Étape 5 — Consigne complète

Inclure :

- but ;
- procédure ;
- structure ;
- critères ;
- vérification.

## Étape 6 — Feedback

Prévoir quand et comment les élèves sauront si leur travail est réussi.

- auto-correction ;
- correction collective ;
- feedback pair ;
- grille critériée ;
- correction commentée.

## Étape 7 — Consolidation

Prévoir une reprise ultérieure.

- J+1 : rappel ;
- J+7 : quiz ;
- J+14 : exercice mélangé ;
- J+30 : transfert.

# 14. Checklists prêtes à l’emploi

## 14.1 Checklist enseignant avant séance

- [ ] L’objectif est formulé clairement.
- [ ] Les prérequis sont identifiés.
- [ ] Les élèves savent ce qui est important.
- [ ] La consigne contient but, procédure, structure, critères, vérification.
- [ ] Une activité cognitive réelle est prévue.
- [ ] Les erreurs probables sont anticipées.
- [ ] Un feedback est prévu.
- [ ] Une trace des essentiels est prévue.

# 15. Erreurs pédagogiques fréquentes à éviter

## 15.1 Confondre exposition et apprentissage

Dire une chose clairement ne garantit pas qu’elle est apprise.

## 15.2 Confondre activité et engagement cognitif

Un élève peut être très occupé mais peu engagé intellectuellement.

## 15.3 Donner des devoirs sans enseigner la méthode

Le travail personnel doit être préparé et réinvesti.

## 15.4 Évaluer sans feedback exploitable

Une note seule ne suffit pas à apprendre.

## 15.5 Surcharger la mémoire de travail

Trop d’informations nouvelles en même temps bloque l’apprentissage.

## 15.6 Croire que relire suffit à mémoriser

La récupération active est plus efficace.

## 15.7 Encourager sans créer de réussite réelle

Le sentiment d’efficacité personnelle se construit par des expériences maîtrisées, pas seulement par des phrases positives.

# 16. Synthèse ultra-courte à retenir

1. L’attention est la porte d’entrée de l’apprentissage.
2. L’élève apprend mieux quand il est cognitivement actif.
3. L’erreur devient utile avec un feedback précis.
4. La mémoire exige des reprises espacées.
5. Les savoirs scolaires sont des connaissances secondaires : ils doivent être enseignés explicitement.
6. Une consigne efficace dit le but, la méthode, la forme, les critères et la vérification.
7. Le travail personnel commence en classe.
8. La différenciation ajuste les chemins sans baisser l’ambition.
9. La motivation dépend beaucoup de la compétence perçue, de la valeur de la tâche, de l’autonomie et du climat.
10. Le sentiment d’efficacité personnelle se construit par des réussites graduées et attribuées à de bonnes stratégies.
